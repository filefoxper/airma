# 安装与支持

当前工具的可运行包维护在[npm](https://www.npmjs.com/get-npm)官方服务器。您可以运行如下命令获取最新的稳定版本。

## 安装命令

```
npm i @airma/react-state
```

## 浏览器支持

```
chrome: '>=58',
edge: '>=16',
firefox: '=>57',
safari: '>=11'
```

如果你需要在更早版本的浏览器使用该工具，需要对其提供全面的 polyfill 支持。
