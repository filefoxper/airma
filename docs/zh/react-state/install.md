# 安装于支持

`@airma/react-state` 包文件存放在 [npm](https://www.npmjs.com/get-npm) 中。可通过以下命令进行安装。

## 安装命令

```
npm i @airma/react-state
```

## 浏览器支持

```
chrome: '>=58',
edge: '>=16',
firefox: '=>57',
safari: '>=11'
```
如果你需要支持更早的浏览器版本，请自行通过 polyfill 进行兼容处理。

进入下一章节可以了解 `@airma/react-state` 的相关 [概念](/zh/react-state/concepts.md)。
